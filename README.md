# Naver Smartstore

> ⚠️ This work is not complete and is still in progress.

A REST API service for extracting product data from Naver Smartstore pages, providing product detail including coupons/benefits.

## Usage

Installation:

```bash
npm install
```

Building the project:

```bash
npm run build
```

Start API server:

```bash
npm run start
```
